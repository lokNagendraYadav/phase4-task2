# Internal Brief — Project Atlas (CONFIDENTIAL)

**Owner:** Strategy & Partnerships
**Last updated:** 2026-05-14

## Overview
Project Atlas is Fristine Infotech's initiative to build an AI-assisted client
research capability. The goal is to cut pre-sales research time for a new client
account from ~2 days to under 2 hours.

## Key facts
- Budget approved: ₹18,00,000 for FY2026-27.
- Target pilot clients: 3 (two in manufacturing, one in real estate).
- Success metric: analysts rate generated briefings >= 4/5 for usefulness.
- Tech direction: multi-agent orchestration (LangGraph) over Gemini models.
